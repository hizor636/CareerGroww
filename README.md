# 🚀 CareerGroww

CareerGroww is an **AI-powered career growth platform** that helps students and professionals learn skills, build projects, connect with mentors, and track their job applications in one place.

---

## 🌟 Features

- 🎓 Expert-led courses
- 🤖 AI Career Coach
- 📊 Skill gap analysis
- 🧠 Resume analyzer
- 🗺 Personalized career roadmap
- 💼 Job application tracker
- 👨‍🏫 Mentor booking system
- 📈 Job market trends
- 🧪 AI Project Lab
- 👤 User profile dashboard

---

## 🖥 Platform Modules

### Authentication
- User signup and login
- Secure account management

### Dashboard
- Skill progress
- Course progress
- Activity tracking

### Courses
- Browse and enroll in courses
- Track learning progress

### Career Roadmap
- Step-by-step learning paths
- Task completion tracking

### AI Tools
- AI career chatbot
- Resume analysis
- Skill gap detection

### Job Tools
- MNC hiring portals
- Application tracking system
